Plana

プライバシーポリシー

最終更新日：2026年6月19日

本プライバシーポリシー（以下「本ポリシー」）は、リオ（成瀬璃雄）（以下「開発者」）が提供するモバイルアプリケーション「Plana」（以下「本アプリ」）における、ユーザーの個人情報の取扱いについて定めるものです。

本アプリは、ユーザー同士がお出かけプランを共有し、写真を共有するためのソーシャルアプリです。ご利用にあたっては本ポリシーをよくお読みいただき、内容にご同意のうえでご利用ください。

1. 開発者情報

  • 開発者：リオ（成瀬璃雄）
  • 連絡先：steplife.contact@gmail.com
  • 所在地：日本国愛知県

2. 取得する情報

本アプリは、サービス提供のために以下の情報を取得します。

2.1 認証情報

  • Apple ID による認証情報（Sign in with Apple）
  • Firebase Authentication が発行する一意なユーザーID（UID）

※ Apple ID のメールアドレスは取得しません。Apple がプライベートリレーアドレスを提供する場合は、それも取得しません。

2.2 ユーザーが入力・登録する情報

  • 表示名（ユーザー自身が設定）
  • 自己紹介文（任意）
  • プロフィールアバター（システム提供のアイコン、またはユーザーが選択した画像）
  • 招待コード（自動生成）
  • フォロー／フォロワー関係
  • お出かけプランの情報（タイトル、日付、説明、行き先、メンバー、表紙アイコン）
  • お出かけアルバムにアップロードされた写真およびキャプション

2.3 自動取得する情報

  • アプリのバージョン情報
  • クラッシュレポート（Firebase 経由、匿名）

3. 情報の利用目的

取得した情報は、以下の目的のために利用します。

  • 本アプリの提供および機能維持
  • ユーザー間でのお出かけプランと写真の共有
  • 招待コードによる友達追加機能の提供
  • フォロー関係に基づく相互限定の共有機能の実現
  • 不具合対応およびサービス改善
  • お問い合わせへの対応

4. 情報の保存場所

本アプリで取得した情報は、Google LLC が提供する Firebase（Google Cloud Platform）のサーバー上に保存されます。具体的には以下のリージョンを利用します。

  • Cloud Firestore：asia-northeast1（東京）
  • Cloud Storage for Firebase：us-east1（米国）
  • Firebase Authentication：Google の標準提供環境

Firebase のセキュリティおよびプライバシーに関する詳細は、Google のプライバシーポリシー（https://policies.google.com/privacy）をご参照ください。

5. 第三者への提供

本アプリは、以下の場合を除き、取得した情報を第三者に提供しません。

  • ユーザー本人の同意がある場合
  • サービス提供に必要な範囲で Google Firebase に保存する場合
  • 法令に基づく要請があった場合

6. 他ユーザーへの開示

本アプリの性質上、以下の情報は他のユーザーに開示されます。

  • 表示名、自己紹介、プロフィール写真／アバター
  • 招待コード（共有された相手のみ）
  • お出かけプランに参加しているメンバー間で共有されるプラン情報および写真
  • 相互フォロー関係にあるユーザー間でのみ表示される情報

7. ユーザーの権利

ユーザーは、本アプリ内の機能により以下を行うことができます。

  • プロフィール情報の閲覧および編集
  • お出かけプランおよび写真の編集・削除
  • フォロー関係の解除
  • サインアウト
  • アカウントおよび関連データの完全削除

アカウント削除を実行すると、ユーザーが作成したお出かけ、アップロードした写真、プロフィール情報、フォロー関係、認証情報がすべて削除されます。この操作は取り消すことができません。

8. データの保持期間

本アプリで取得した情報は、ユーザーがアカウントを削除するまで保持されます。アカウント削除後は速やかにデータを削除しますが、バックアップ等の都合により完全削除に最大30日程度を要する場合があります。

9. セキュリティ

取得した情報は、Firebase のセキュリティルールおよび HTTPS 通信により保護されています。ただし、インターネット通信および電子的記録の完全な安全性を保証するものではありません。

10. 未成年者の利用

本アプリは13歳以上の利用を想定しています。13歳未満のお子様は、保護者の同意のもとご利用ください。

11. プライバシーポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。重要な変更がある場合は、本アプリ内またはこのページにてお知らせします。
